# Tech Stack - E-commerce Order Management Service

## Core Environment
- **Runtime:** Node.js
- **Framework:** Express.js
- **Language:** JavaScript (ES6+)

## Persistence
- **Database:** MongoDB
- **ODM (Object Data Modeling):** Mongoose (for schema-based modeling and validation)

## Messaging
- **Message Broker:** RabbitMQ
- **Client Library:** amqplib

## Testing
- **Framework:** Jest (used for unit testing and coverage reporting)

## Infrastructure & DevOps
- **Containerization:** Docker
- **Orchestration:** Docker Compose
- **Git Hooks:** Husky, lint-staged
- **Linting & Formatting:** ESLint, Prettier
